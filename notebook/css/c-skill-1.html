    CSS 背景图片

    Mr.Sunny 5/19/2021 CSSSkill

    好看的网页，其背后离不开好看的背景图，下面记录下对背景图的处理技巧！！！

    设置背景图

    设置方式：background-image: url()

    1. 图片的地址必须放在 url() 中，图片的地址可以是本地的地址，也可以是网络的地址
    2. 如果图片的大小没有标签的大小大，那么会自动在水平和垂直方向平铺来填充
    3. 如果网页上出现了图片，那么浏览器会再次发送请求获取图片

    控制平铺方式

    控制方式：background-repeat

    取值：

    • repeat：默认，在水平和垂直都需要平铺
    • no-repeat：在水平和垂直都不需要平铺
    • repeat-x：只在水平方向平铺
    • repeat-y：只在垂直方向平铺

    应用：可以通过背景图片的平铺来降低图片的大小，提升网页的访问速度

    控制背景图位置

    控制方式：background-position

    取值：具体的方位名词

    • 水平方向：left center right
    • 垂直方向：top center bottom

    取值：具体的像素，如：background-position: 100px 200px;

    1. 记住一定要写单位
    2. 支持负数
    3. 当前容器进行定位，向右为 x 正轴 ，向下为 y 正轴

    背景关联方式

    默认情况下背景图片会随着滚动条的滚动而滚动，如果不想让背景图片随着滚动条的滚动而滚动，那么就需要修改背景图片和滚动条的关联方式

    修改方式：background-attachment

    取值：

    • scroll：默认值，会随着滚动条的滚动而滚动
    • fixed：不会随着滚动条的滚动而滚动

    背景属性缩写

    格式：background: 背景颜色 背景图片 平铺方式 关联方式 定位方式;

    background 属性中，任何一个属性都可以被省略

    背景图片和插入图片区别

    1. 背景图片仅仅是一个装饰，不会占用位置，而插入图片会占用位置
    2. 背景图片有定位属性，可以很方便的控制图片的位置，而插入图片没有
    3. 插入图片的语义比背景图片的语义要强，如果图片想被搜索引擎收录，那么推荐使用插入图片

    扩展：精灵图

    CSS 精灵图是一种图像合成技术，可以减少请求的次数，以及可以降低服务器处理压力，其使用方式需要配合背景图片和背景定位来使用

    1. 指定一个背景区域（Div），该区域（精灵图可视区域）的宽高定为你需要显示的 精灵图中的某一个背景图的宽高
    2. 利用 背景定位 将该精灵图需要显示的那部分定位到你的 可视区域

    技巧和一个小 Demo

    某一张背景图片，是由多张背景图片组合（拼接）而成的，那么处理的技巧就是 div 的嵌套搭配背景图片的定位

    <div class="box1">
  <div class="box2"></div>
</div>

<style>
.box1{
  width: auto;
  height: 768px;
  background-image: url('https://cdn.jsdelivr.net/gh/HaoQiSpace/cdn/demo/images/buyu_bg.jpg');
}
.box2{
  /**
   * 这里需要设置盒子的宽高和外出盒子一样，原因请查看：控制背景图位置
   * - width: 一般是具体的值，这里作为容器适应
   */
  width: auto;
  height: 768px;
  background-image: url('https://cdn.jsdelivr.net/gh/HaoQiSpace/cdn/demo/images/buyu_bottom.png');
  background-position: center bottom;
  background-repeat:no-repeat;
}
</style>
